## What is it?

Online Course App using React Anative

## Tech Stack

1. React Native CLI
2. Apollo Client
3. Hygraph (hygraph.com)
4. Zustand
5. Nativewind (Tailwind css for React Native)
6. React Native Youtube

## How To Run

```bash
# install package
yarn install

# run to android
yarn android
```

## Notes

Dont forget to setup credentials in .env
